# Social Media Web Application (PHP + MySQL)

A dynamic and responsive **Social Media Web Application** built using **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. This platform allows users to connect, create posts, manage profiles, follow/unfollow each other, and interact in real-time â€” similar to a mini version of Instagram or Twitter.

---

## Features

- User Registration & Secure Login
- Profile Management (Full Name, Bio, Profile Picture, etc.)
- Create, View, and Delete Posts
- Send, Accept & Reject Follow Requests
- Search Users by Name or Email
- Real-Time Feed Updates
- Upload Profile Pictures
- Session Management (Login-required pages)
